# 🎨 Introducing SyncCanvas

### The AI-Powered Real-Time Collaborative Whiteboard

---

## ✨ What is SyncCanvas?

SyncCanvas is an infinite multiplayer whiteboard where teams can brainstorm, design, plan, and collaborate in real time.

Think:

> **Miro + Excalidraw + Figma + AI Copilot**

all in one workspace.

### 🌍 Real-Time Collaboration
